## Tools for BAP Servicing
1. The [BAP Deployment Toolkit](https://github.com/bitergia-analytics/bap-deployment-toolkit)
   is a set of tools for provisioning, configuring, and deploying the Bitergia Analytics Platform
   with Terraform and Ansible.
1. The [BAP Operations Toolkit](https://github.com/bitergia-analytics/bap-operations-toolkit) is a set of tools to operate and manage instances of the Bitergia Analytics Platform
1. [BAP documentation for the handover process](https://github.com/bitergia-analytics/documentation-handover)
